Simple step for running

## Download dataset
Go to this link: https://www.kaggle.com/datasets/ronakgohil/license-plate-dataset

Download the dataset, upzip it into the destination folder


## 1. create virtual environment (optional) 
```bash
python -m venv venv

venv\Scripts\Activate.ps1
```

## 2. download the dependencies
```bash
pip install -r requirements.txt
```

## 3. run
```bash
python ocr.py
```
